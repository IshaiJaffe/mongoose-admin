Mongoose-Admin

A front end to Mongoose - the ORM for Node.js and MongoDB

var admin = require('mongoose-admin').createAdmin();
admin.registerModel(BlogPostFields);

Features

  1. Automatic forms
  2. Supports nested objects
  3. Uses only public mongoose apis
